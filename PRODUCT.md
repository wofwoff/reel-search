# Product

## Register

product

## Users

People who save useful Instagram reels, YouTube videos, and media references and want to recover the underlying idea later instead of maintaining a manual bookmark taxonomy. The primary workflow is saving quickly, then browsing or semantically searching a private cross-device library.

## Product Purpose

ReelMind turns short-form media into a durable, searchable knowledge library. Success means saving remains one-step simple, the real archive size is trustworthy, and collections describe what content is about rather than the platform, repository, or AI tool used to make it.

## Brand Personality

Focused, warm, and quietly intelligent. The interface should feel calm enough for personal reflection while remaining precise and practical as a daily tool.

## Anti-references

- Oversized brand artwork that competes with the save action.
- Generic analytics dashboards or dense walls of identical reel cards.
- Collections named after incidental sources such as GitHub or Claude Code when the actual subject is design, editing, cooking, singing, or another craft.
- Decorative motion, excessive shadows, nested cards, and low-contrast muted text.

## Design Principles

1. Saving is the first action and must remain immediately available.
2. Organize by human meaning: broad domain first, specific topic second.
3. Show enough preview content to build recognition without loading the whole archive.
4. Preserve continuity across browser, phone, and iPhone Shortcut workflows.
5. Use progressive disclosure: collections first, individual reels after search or collection selection.

## Accessibility & Inclusion

Target WCAG 2.1 AA contrast, visible keyboard focus, semantic controls, useful alternative text, responsive layouts down to 320px, and reduced-motion support.
